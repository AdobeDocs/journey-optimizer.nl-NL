# trim {#trim}

Tekenreeks

## Functiesyntaxis

`trim(<parameters>)`

## Parameter

| Parameter | Type |
|-----------|------------------|
| string | string |

## Handtekening en type geretourneerd

`trim(<string>)`

Retourneer een tekenreeks.
