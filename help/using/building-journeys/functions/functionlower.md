# lower {#lower}

Retourneert een versie in kleine letters van de parameter.

## Categorie

Tekenreeks

## Functiesyntaxis

`lower(<parameter>)`

## Parameter

* string

## Handtekening en type geretourneerd

`lower(<string>)`

Retourneert een tekenreeks.

## Voorbeeld

`lower("A")`

Retourneert &quot;a&quot;.
