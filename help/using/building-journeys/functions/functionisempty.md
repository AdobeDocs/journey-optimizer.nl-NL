# isEmpty {#isEmpty}

Retourneert true als de tekenreeks in de parameter geen teken heeft.

## Categorie

Tekenreeks

## Functiesyntaxis

`isEmpty(<parameters>)`

## Parameters

* string

## Handtekening en type geretourneerd

`isEmpty(<string>)`

Retourneert een Booleaanse waarde.

## Voorbeeld

`isEmpty("")`

Retourneert true.

`isEmpty("Hello World")`

Retourneert false.
