# Ingebouwde machtigingen {#ootb-permissions}

Ingebouwde machtigingen vertegenwoordigen de verschillende machtigingen die aan een **[!UICONTROL Role]** kunnen worden toegewezen om de gebruikerstoegang tot Journey Optimizer te verfijnen. De toestemmingen op hoog niveau omvatten laagvlakke toestemmingen, die op [&#x200B; worden gedetailleerd deze pagina &#x200B;](high-low-permissions.md).
